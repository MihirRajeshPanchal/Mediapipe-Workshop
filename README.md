## Project Setup

### Next.js Face Mesh Detection Mediapipe JS

```
cd next
npm install
npm run dev
```


### Streamlit Hand Gesture Recognition

```
cd streamlit
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
